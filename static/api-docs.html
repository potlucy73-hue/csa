Introduction

The Carrier Data Management System (CDMS) API provides programmatic access to carrier data from the FMCSA database. This API allows you to lookup, search, and batch process carrier information. 

Base URL: https://api.cdms.com/v1

Authentication

All API requests require authentication using an API key. Include your API key in the request header: 

Authorization: Bearer YOUR_API_KEY

Lookup Carrier

Retrieve detailed information about a specific carrier using their MC number. 

GET /carriers/{mc_number}

Example Response

{
  "mc_number": "123456",
  "company_name": "Example Trucking LLC",
  "dot_number": "987654",
  "operating_status": "AUTHORIZED",
  "insurance": {
    "status": "ACTIVE",
    "expiry_date": "2024-12-31"
  },
  "safety_rating": "Satisfactory",
  "address": {
    "street": "123 Main St",
    "city": "Springfield",
    "state": "IL",
    "zip": "62701"
  }
}

Batch Processing

Process multiple MC numbers in a single request. 

POST /carriers/batch
Content-Type: application/json

{
  "mc_numbers": ["123456", "234567", "345678"],
  "options": {
    "include_safety_data": true,
    "include_insurance": true
  }
}

Search Carriers

Search for carriers using various criteria. 

GET /carriers/search?state=IL&safety_rating=satisfactory
